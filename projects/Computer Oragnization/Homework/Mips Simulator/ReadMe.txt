1. MIPS_Simulator.c is the code of the MIPS simulator.

2. MIPS Simulator Handbook.pdf is the handbook of the MIPS simulator.

3. mips.txt and code.txt are the test data:
	mips.txt contains the MIPS program.
	code.txt is the machine code of the test program.
	
4. You can run the MIPS simulator by the MIPS_Simulator.exe directly.
